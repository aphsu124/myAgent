一、市場概覽

昨日全球市場情緒複雜，區域股市表現不一。值得關注的是，印尼市場錄得顯著的外資淨賣出，單日淨流出達2,100.1億印尼盾，本年度至今累計淨流出更達7.02267兆印尼盾，反映資金正持續流出新興市場，可能對區域情緒造成壓力。儘管部分股票在特定因素驅動下有所表現，但整體市場氛圍仍需警惕資金流向變化。

二、棕櫚油市場動態

棕櫚油期貨市場近期表現活躍，此前曾一度觸及15個月新高，主要受到印尼可能推進其棕櫚油生物柴油政策的預期所驅動。印尼作為全球最大的棕櫚油生產國，其政策動向對全球供應與價格具有舉足輕重的影響。然而，市場在達到高峰後，也出現部分獲利回吐。基準6月交割的棕櫚油合約在馬來西亞衍生品交易所（BMD）最新報價為每公噸4,498令吉，較此前高位有所回落，顯示市場在經歷快速上漲後，已進入盤整階段，投資者正權衡未來供需及政策變數。

三、馬來西亞股市觀察

馬來西亞股市方面，整體走勢缺乏明確方向，個股表現分歧。儘管市場上不乏併購消息（如Sunway對IJM的收購案，價值達110億令吉或每股3.15令吉）及特定公司表現，但整體市場仍受區域資金流動和商品價格波動影響。部分如Petron Malaysia等公司股價波動不大，而特定種植股則可能受棕櫚油價格影響。投資者宜繼續關注企業財報及宏觀經濟數據，以評估市場走向。

四、展望

展望未來，棕櫚油價格的關鍵驅動因素將持續圍繞印尼的出口政策調整及其國內需求。任何政策變動，尤其是生物柴油摻混比例的提升，都將對全球供應與價格產生即時影響。區域股市則需警惕外資流動性變化，其對整體市場情緒的影響不容忽視。建議投資者密切關注宏觀經濟指標、主要商品價格走勢以及地緣政治風險，以做出審慎的投資決策。
