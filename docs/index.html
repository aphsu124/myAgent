📰 泰國棕櫚油晨間新聞

晨間新聞 2026年3月31日

各位投資者早安，我是您的資深分析師。以下是今日市場的重點觀察。

棕櫚油市場動態

馬來西亞衍生產品交易所（BMD）的基準6月棕櫚油期貨合約在昨日（3月30日）收盤時表現強勁，上漲36令吉，漲幅0.78%，收報每公噸4,667令吉，延續了近期的上漲勢頭。在上週，棕櫚油期貨已連續第四週實現上漲，週五（推測為3月27日）更攀升47令吉，漲幅1.03%，達到每公噸4,630令吉，顯示市場情緒樂觀。

最新的利好消息來自泰國政府，該國宣布擴大B20生物柴油的使用範圍。此舉不僅旨在降低國內燃油成本，更關鍵的是透過增加對當地棕櫚作物的需求，穩定泰國的棕櫚油市場。這項政策預計將為區域棕櫚油需求帶來正面提振，為價格提供堅實支撐。

儘管在上週中期（約3月25日）曾因競爭性食用油和原油價格走軟而小幅回落，但隨後在3月26日（星期四）迅速反彈超過1%，主要受到芝加哥豆油和有利市場數據的支撐，顯示市場整體看漲情緒依然濃厚。

企業新聞焦點

在企業層面，市場聚焦於數宗重要事件：

  • 併購動態：建築巨頭金務大（IJM）與陽光集團（Sunway）之間價值110億令吉的收購案取得重要進展。市場投票已通過陽光集團於1月12日提出的每股3.15令吉的收購要約，這標誌著馬來西亞建築與房地產行業的一宗重大併購案塵埃落定。
  • 營運挑戰：Petron Malaysia（馬來西亞國家石油精煉及銷售公司，股號3042）表示，其波德申煉油廠碼頭事件對2026年的資本支出影響仍在預期中，目前海床殘骸清理工作持續進行，這將是今年需要關注的營運風險點。

總結與展望

總體而言，泰國生物柴油政策的積極信號，加上芝加哥豆油等外部因素的支撐，預計將為棕櫚油市場提供堅實的基礎。儘管全球原油價格和競爭性食用油的波動仍需密切關注，但需求面傳來的利好消息有望穩定市場信心。投資者應持續留意宏觀經濟數據及相關政策動態。
