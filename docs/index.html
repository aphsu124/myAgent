FFB與CPO趨勢分析

目前泰國國內鮮果串(FFB)價格報每公斤8.1泰銖，而原棕櫚油(CPO)價格則維持在每公斤45.5泰銖的高位。這些價格顯示泰國國內市場需求依然強勁，且供應面可能存在一定支撐。儘管國際原棕櫚油期貨市場近期有所波動，泰國國內價格仍展現出較強的韌性。

從歷史趨勢來看，FFB價格在過去一個月維持在高檔區間，主要受到國內壓榨廠為滿足持續成長的食用油及生物柴油產業需求而積極採購所推動。CPO價格的高企則反映了整體供應相對偏緊的局面，尤其是在即將進入生產旺季之際，市場對產量能否快速回升存在觀望情緒。此價格水平對於棕櫚果農而言，提供了良好的收益基礎。

BMD期貨與匯率對基差的影響解讀

今日馬來西亞衍生產品交易所(BMD)的原棕櫚油期貨基準價格為每噸4828馬來西亞令吉。同時，馬來西亞令吉兌泰銖的匯率為1馬來西亞令吉兌7.78泰銖。將BMD期貨價格換算為泰銖後，約為每噸37,578.24泰銖 (4828 MYR * 7.78 THB/MYR)。相較於泰國國內每噸45,500泰銖 (45.5 THB/kg)的原棕櫚油現貨價格，我們觀察到泰國市場存在顯著的正基差，約為每噸7,921.76泰銖。

此一龐大正基差，明確指示泰國國內原棕櫚油價格顯著高於國際市場（經匯率調整後）。這可能反映了多重因素的綜合作用，包括泰國國內供應的相對緊俏、強勁的本土需求未完全被外部供應所滿足，以及潛在的進口障礙或較高的運輸成本考量。馬來西亞令吉的相對走軟，雖然在一定程度上支持了BMD期貨價格，使其對非馬來西亞買家更具吸引力，但對於泰國而言，卻可能使得從馬來西亞進口原棕櫚油在泰銖計價後顯得相對便宜。然而，目前此種價格壓力尚未完全傳導至泰國現貨市場，顯示泰國市場的獨立性與特殊性在短期內仍佔主導地位。
