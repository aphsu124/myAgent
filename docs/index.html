各位同仁，早安。今日泰國棕櫚油市場在多重因素交織下呈現穩定偏強的格局。鮮果串（FFB）價格維持在每公斤8.1泰銖，毛棕櫚油（CPO）價格為每公斤45.5泰銖。同時，馬來西亞衍生品交易所（BMD）的指標毛棕櫚油期貨（BMD CPO）報價為每噸4828馬來西亞令吉，而馬來西亞令吉兌泰銖匯率為1:7.78。

FFB與CPO趨勢分析：泰國國內FFB價格（8.1 THB/kg）與CPO價格（45.5 THB/kg）維持在健康水平。換算下來，一噸毛棕櫚油約需220公斤的FFB，顯示加工商的利潤空間相對穩定。國內CPO價格相較於國際期貨市場表現出較強的獨立性與溢價，這通常意味著國內供需關係緊張或政策支持力度較大。雖然全球棕櫚油價格近期波動，但泰國國內市場由於其獨特的結構，價格受外部傳導的影響有時會滯後或表現出差異。目前FFB價格堅挺，反映了加工廠對鮮果串的穩定需求，這可能預示著未來幾週CPO產量將保持穩定或略有增加，但也預示著原料成本的支撐作用。

BMD期貨與匯率對基差的影響解讀：將BMD CPO期貨價格轉換為泰銖計價：4828 MYR/噸 * 7.78 THB/MYR ≈ 37575.84 THB/噸。而泰國國內CPO價格為45.5 THB/kg，即45500 THB/噸。由此可見，泰國國內CPO價格（45500 THB/噸）較BMD期貨換算價格（37575.84 THB/噸）高出約7924.16 THB/噸，形成了顯著的「正基差」。這種高基差現象凸顯了泰國國內棕櫚油市場的獨特動態。儘管馬來西亞令吉走軟理論上會降低馬來西亞棕櫚油的泰銖進口成本，但目前如此高的正基差表明，泰國國內市場有著強勁的內生需求或面臨供應瓶頸，足以抵消部分匯率帶來的潛在進口優勢。

關鍵影響因素分析：

  1. 國內供應與需求失衡： 泰國國內毛棕櫚油價格顯著高於國際期貨換算價格，顯示國內市場存在強勁的需求或供應緊張。這可能與泰國政府的生物柴油強制摻混政策有關，該政策為國內毛棕櫚油創造了穩定的基本需求，並可能導致加工商優先滿足國內市場。若目前處於產量恢復期但尚未達到高峰，也可能導致供應暫時吃緊。
  2. 國際原油與油脂市場支撐： 儘管馬來西亞衍生品交易所（BMD）毛棕櫚油期貨近期有所波動，但整體維持在相對高位（4828 MYR/噸）。這反映了全球植物油市場，特別是原油價格的支撐（例如先前提到「美國-伊朗衝突」推升原油價格）。棕櫚油作為可再生能源的原料之一，其價格容易受到原油走勢及其他主要植物油（如大豆油）供需的影響，共同維持了全球棕櫚油價格的底部。
  3. 匯率因素與進口成本： 馬來西亞令吉兌泰銖的匯率（1 MYR = 7.78 THB）是影響泰國進口棕櫚油成本的關鍵。雖然參考資料顯示令吉近期相對疲軟，這理論上會使馬來西亞棕櫚油對泰國買家而言更便宜，從而可能縮小基差。然而，目前高昂的基差表明，即使令吉走軟，泰國國內市場的供需壓力仍然足以抵消部分進口誘因，或與高昂的進口關稅及物流成本有關，使得國內價格難以下跌。
  4. 季節性生產與庫存水平： 棕櫚油生產具有明顯的季節性。雖然目前已進入4月，產量通常會逐步增加，但如果前一季度的惡劣天氣或病蟲害影響了鮮果串的生長，或者現有庫存水平較低，都可能限制市場供應。低庫存會增加現貨市場的競價壓力，進而推高FFB和CPO的價格，這是造成高基差的重要內部因素。

綜合來看，泰國棕櫚油市場在穩定的國內需求和政策支持下，展現出較強的韌性。儘管國際市場存在波動，但國內FFB與CPO價格仍保持健康，且基差顯著，預計短期內這種強勢格局將持續。市場參與者需密切關注國內生產數據、政府政策調整以及國際原油與油脂市場的最新動態。
